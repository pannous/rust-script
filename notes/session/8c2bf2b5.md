## Implementation

Added the `is` operator which provides Python-style runtime type checking:

```rust
x := 1
x is i32        // true
x is String     // false

str := "hello"
str is String   // true

r := 'a'
r is rune       // true (rune = char)

let y: i64 = 42
y is int        // true (int = i64)
```
